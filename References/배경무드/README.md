# 배경 무드 레퍼런스

> 기획서 v8-2 (Stage별 배경 무드 다양화) 참고 자료.
> Unity 빌드 비포함 (Assets/ 외부 폴더).

## 컨셉

스테이지 = 나이대 = 인생 시기. 각 시기에 어울리는 **장소·소품·분위기**로 배경 4레이어(하늘 / 주변 / 테이블 / 보드)를 차별화.

- **하늘**: 시간대·날씨·계절감
- **주변**(테이블 너머 공간): 장소 — 캠퍼스, 사무실, 거실, 요양원 등
- **테이블**: 재질 / 색 / 마모도
- **보드(매트)**: 색 / 패턴 (단, 매트 영역 = `BoardBounds` 정의는 텍스처와 무관)

## 활용 방식

1. v8-2 무드 결정 세션 시 base 레퍼런스.
2. 에셋 생성(OpenAI/Gemini) 시 `--ref` 인자 또는 프롬프트 기술 참고.
3. 픽셀 아트 톤(`완성톤_픽셀아트`)을 채택할지, 손그림 톤을 채택할지는 별도 결정.

## 폴더 규칙

- **덮어쓰기 금지** (donts/images.md). 새 안은 `_v2`, `_v3` suffix.
- 파일명 형식: `{나이}살_{메카닉}_{설명}.png`
- 픽셀 아트 vs 손그림은 suffix(`_픽셀아트`, `_스케치`)로 구분.
