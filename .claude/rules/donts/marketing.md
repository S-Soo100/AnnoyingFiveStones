# Don'ts — 마케팅 / 콘텐츠 발행

> 콘텐츠 발행/마케팅 작업 시 반복된 실수. 루트 [`donts.md`](../donts.md)의 제너럴 규칙과 함께 적용.

## 🔗 플랫폼별 링크

1. **LinkedIn에 한글 URL 직접 붙이기 금지** — 한글 깨짐/인코딩 문제로 링크 안 열림. 반드시 `bit.ly` 등 단축 URL 사용.

## 📁 폴더 구조

2. **사업계획서는 별도 폴더 분리** — ideaBank 루트/카테고리 폴더에 섞어 넣지 말 것. 전용 폴더(`proposals/` 등)로 분리해서 관리.

## 🧑‍💼 에이전트 순서

3. **편집장 → 콘텐츠 어댑터 → 브랜드 매니저 순서 지키기** — 스캔/추천은 편집장(marketing-director), 변환은 content-adapter, 발행 전 검수는 brand-manager. 순서 섞으면 톤/브랜드 일관성 깨짐.

## ✍️ 발행 전

4. **브랜드 매니저 검수 건너뛰기 금지** — 톤·문체·브랜드 체크 없이 발행 금지. 특히 Velog/LinkedIn/Threads처럼 페르소나가 다른 채널은 필수.
